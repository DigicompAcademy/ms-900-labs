# Schritt-für-Schritt-Anleitung: SharePoint Teamwebsite erstellen

## ✅ 1. Anmelden
- Gehe auf https://www.office.com.
- Melde dich mit deinem Microsoft 365 Firmenkonto an.

## ✅ 2. SharePoint öffnen
- Klicke oben links auf das **App-Startfeld (neun Punkte)**.
- Wähle **SharePoint** aus.

## ✅ 3. Neue Teamwebsite erstellen
- Im SharePoint-Portal klicke auf **Website erstellen**.
- Wähle **Teamwebsite** (für Zusammenarbeit im Team).
- Gib einen **Namen** für die Website ein (z. B. „Marketing-Team“).
- Optional: Füge eine **Beschreibung** hinzu.
- Wähle die **Datenschutz-Einstellung**:
  - **Privat** (nur Mitglieder haben Zugriff).
  - **Öffentlich** (alle in der Organisation können zugreifen).
- Klicke auf **Weiter**.
- Füge **Mitglieder** hinzu (Kollegen, die Zugriff haben sollen).
- Klicke auf **Fertigstellen**.

## ✅ 4. Inhalte hinzufügen
- Nach der Erstellung öffnet sich die Teamwebsite.
- Du kannst:
  - **Dokumentbibliothek** nutzen (Dateien hochladen).
  - **Listen** erstellen (z. B. Aufgaben, Kontakte).
  - **Seiten** hinzufügen (für interne Infos).
- Passe die Navigation und das Layout an.

## ✅ 5. Seite veröffentlichen und teilen
- Klicke auf **Veröffentlichen**, wenn du Änderungen abgeschlossen hast.
- Teile den Link mit deinem Team.
